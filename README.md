# skills
A specialised repository of modular skills and scripts. Crafted in Markdown to inject robust domain knowledge and behavioural rules directly into AI coding agents. In agentic workflows compiled entirely in Markdown (`.md`). 

These files are designed to inject robust domain knowledge, behavioural rules, and specific technical capabilities directly into AI coding assistants (like Antigravity , Claude CLI, or custom LLM agents).

---

## 🧩 What is an "AI Skill"?

In modern development, treating Markdown context as infrastructure allows us to build predictable, high-performing AI workflows. Each `.md` file in this repository acts as a "skill block" or system prompt that teaches an LLM exactly how to behave, write code, or execute tasks within a specific domain.
